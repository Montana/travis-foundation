Open Source Funds

There are so many great ideas and creative developers in our community, we think should be supported. A wonderful idea is only the beginning and we want to help with one of the big next steps: financial support. If it's a young project that needs help being kickstarted or an established project to be continued: We want to help expetional Open Source projects by supporting developers and organize funding.
